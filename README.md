<br/></br>### 课程介绍:
MVVM代表一种应用程序的架构模式或开发模式，在最近这些年非常流行。这套课程的目的主要是帮助大家能够掌握基于XAML应用程序的MVVM开发。
课程不仅包含全面细致的理论介绍，并且每一小节后也会有实践，通过实践能够进一步加深对理论部分的理解，并且实践部分还逐步地介绍如何从零开始构建一个MVVM应用程序。
